# Bible Games Dev

Start the Bible Games development server at ~/projects/bible-games. This is a collection of Bible-themed games including Jeopardy and Family Feud variants.

Steps:
1. Navigate to ~/projects/bible-games
2. Check if server is already running on port 3000
3. Install dependencies if needed: npm install
4. Start development server: npm run dev
5. Confirm server started
6. Show URL: http://localhost:3000

Games included:
- Bible Jeopardy: Quiz game with biblical categories
- Family Feud: Bible edition with survey-style questions
- Trivia games with various biblical topics

Tech stack:
- React or Next.js framework
- Interactive game mechanics
- Question/answer database
- Scoring system
- Multiplayer support (optional)

Key features:
- Multiple game modes
- Biblical question categories
- Responsive design for different devices
- Score tracking
- Timer functionality

Perfect for:
- Church youth groups
- Sunday school classes
- Bible study groups
- Family game nights
